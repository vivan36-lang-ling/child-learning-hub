# Session rules

- Never modify layout or styling unless explicitly asked.
- If I think something should change, suggest it first — do not act without approval.
- Do not touch #app-header, screen header rows, or any positioning/sticky/flex properties unless the user explicitly requests it.
- Trust the user's report of what they see — they know their file better than I do.
- The user can also be wrong or confused. If I am sure about something, speak up respectfully and don't blindly agree. Stand by what I know to be true.
